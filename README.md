# Bpm_host

###Review:
Grape mounted on rails API + simple parser for directories with MP3 files

Front-end client: [Bpm_front](https://github.com/Crashtown/bpm_front)

###Documentation:
open
> localhost:3000/apidoc

for grape-swagger API documentation
###Tests:

`

* rails (4.2.0)
* grape (0.11.0)

* ruby-mp3info (0.8.5)
* rspec (3.2.1)
